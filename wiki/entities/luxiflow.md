## Description

Luxiflow is a one-person AI-powered web development agency. It designs and builds custom, high-performance websites for founders and SMBs, using AI to deliver faster without cutting corners on craft. Delivered in 14 days. The website itself is the primary case study in this knowledge base — three separate QA passes have been run across two different ZIP downloads from Lovable.dev.

## Key Facts

- **Stack:** React 18, TypeScript, Vite, Tailwind CSS, Framer Motion, EmailJS [[luxiflow-qa-session-2026-04-30]]
- **Price point (original):** €1,500–€3,500 [[luxiflow-qa-session-2026-04-30]]
- **Price point (v2 ZIP):** €299 Starter / €599 Professional — possible reprice or different variant [[ai-crafted-qa-session-2026-04-30]]
- **Delivery promise:** 14 days from brief to launch
- **Design system:** CSS custom properties (`--gold`, `--bg`, `--surface`, `--elevated`, `--border-c`, `--text`, `--mid`, `--low`)
- **Brand colors:** Gold (`#B8962E`), off-white background (`#F7F7F5`), dark text (`#141413`)
- **QA scores (pass 1):** Technical 9/10, UI/UX 9/10, Conversion 8/10 [[luxiflow-qa-session-2026-04-30]]
- **QA scores (pass 3):** Technical 9/10, UI/UX 9/10, Conversion 8/10 [[ai-crafted-qa-session-2026-04-30]]
